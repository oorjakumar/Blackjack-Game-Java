# Blackjack-Game-Java
AP Computer Science A project: a Java Blackjack game demonstrating OOP concepts

## Project Structure
- **BlackJack.java** – main class that runs the game, combining all other classes
- **BlackJackCard.java** – represents a playing card with Blackjack-specific rules
- **Card.java** – generic card representation (suit, value)
- **Deck.java** – represents the deck of cards
- **Player.java** – represents the human player
- **Dealer.java** – represents the dealer
